将 Auto_Box_Update.img、carplay.conf、carplay.png 
三个文件放在U盘根目录，U盘插入盒子即可自动升级。

文件说明：
    1. carplay.png  是需要修改的汽车图标文件。注意改文件大小不要超过100Kb，分辨率为180x180。
    2. carplay.conf 是修改文字说明的配置文件。只需要修改 “oemIconLabel = ” 后面的值即可。




Put Auto_Box_Update.img, carplay.conf, carplay.png
Three files are placed in the root directory of the USB flash drive, and the USB flash drive is inserted into the box to be automatically upgraded.

Document description:
     1. carplay.png is the car icon file that needs to be modified. Note that the file size should not exceed 100Kb and the resolution is 180x180.
     2. carplay.conf is the configuration file for modifying the text description. Just modify the value after "oemIconLabel = ".
